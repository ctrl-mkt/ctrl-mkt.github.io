AI 카드 수준 차이

기본 수준 (Free/Basic)

제공 항목: summary + analysis

summary: "매출 9.9% 증가"

analysis: "배민 채널이 78% 기여했습니다. 전주 대비 좋은 성과입니다." (Comment: 기본버전에서는 지원 안하는게 맞지 않나 함, 아니면 AI카드를 아예 빼던가, 하나만 띄워주던가)

  • 숫자 + 간단한 해석
  • 제안(proposal) 없음
  • 질문(questions) 없음

상세 수준 (Deluxe 이상)

제공 항목: summary + analysis + proposal + questions

summary: "매출 9.9% 증가, 4주 중 1-2번째로 좋음"

analysis: "배민 채널이 78% 기여했으나 의존도가 높아 다변화 필요..."

proposal: "1. 인기 메뉴 가격 10% 인상
2. 배민 광고비 20% 증액..."

questions: "배민에서 이벤트를 진행했나요?"

  • AI 해석 + 구체적 액션
  • 실행 가능한 제안 2-3가지
  • 외부 요인 확인 질문 3-4개
